This is Amazon Wishlist Calendar

公開された Amazon のほしい物リストに含まれる商品の発売日予定を出力します
対応している形式は iCal および RSS フィードです
一度作成すればほしい物リストが更新されれば自動的に更新されます

試す

Amazon でほしい物リストを確認する

Amazon のほしい物リスト を開いて利用したいリストをクリックします

次へ

ほしい物リストの Id を送信する

https://www.amazon.co.jp/gp/registry/wishlist/{wishlistId}/ref=cm_wl_list_o_1?
上のような URL から wishlistId を抜き出し入力し、送信してください。

無効な wishlistId です
